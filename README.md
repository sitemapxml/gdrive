# gdrive
 Skripta za preuzimanje fajlova sa google drive servisa

# Pokretanje

```
git clone https://github.com/sitemapxml/gdrive.git
cd gdrive
chmod +x gdrive
./gdrive
```

Nakon pokretanja fajla potrebno je uneti adresu za preuzimanje i naziv fajla sa ekstenzijom.
